# Hospital Triage & Appointment System 

This repository contains a single-file Java implementation of the Hospital Appointment & Triage System required for the Data Structures capstone assignment.

## Files
- `src/HospitalSystem.java` : Complete Java implementation (compile and run as explained below).
- `sample_output.txt` : Example interaction outputs (sample inputs/outputs).
- `report.md` : 2-page technical report (ADT sketches, flow, complexity).

## How to compile and run
Open a terminal with Java installed:
```bash
cd src
javac HospitalSystem.java
java HospitalSystem
